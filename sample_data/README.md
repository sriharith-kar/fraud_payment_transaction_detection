# Sample Data

These files are small synthetic examples for documentation and quick local testing. They are not real customer records and should not be used as production training data.

## Files

- `european_creditcard_sample.csv`: PCA-style credit card transaction sample with a fraud label.
- `synthetic_payment_sample.csv`: readable payment transaction sample with merchant, amount, device risk, and fraud label fields.

The full datasets are intentionally excluded from GitHub because of size and licensing considerations.
